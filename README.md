# NCTU OJ CLI

CLI for [NCTU OJ](https://oj.nctu.me)

## Installation

```sh
$ git clone https://github.com/c910335/nctu-oj-cli.git
$ cd nctu-oj-cli
$ shards build
$ cp config.sample.yml config.yml
$ vim config.yml
```

## Usage

```sh
$ bin/nctu_oj scoreboard
```
